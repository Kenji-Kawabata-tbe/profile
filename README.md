    - 負荷状況に応じたスケールアウト、スケールインの設定

    - AppMeshよりも管理コストが低減できる

  - CI／CD設定

  - データベースのマイグレーションを行うGithub　Actionsのワークフロー作成
- 新機能検討
  - ビジネスメンバーと今後必要な機能を適宜検討
